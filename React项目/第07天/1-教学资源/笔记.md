## 完成吸顶效果
	思路分析：
		1、监听滚动
		2、判断滚动出去的距离？（Affix组件中的占位符组件）
		3、如果占位符组件已经滚动出去了
			让内容组件固定布局（脱离标准流）
			让占位符组件撑开（让高度和内容组件高度一致）
		4、如果占位符滚回来（用户可以在可视区域看到他）
			让内容组件去除固定布局（再次回到标准流中）
			让占位符组件高度为0

## websocket(socket.io)
	https://www.runoob.com/html/html5-websocket.html

	HTTP
		短连接

		优点：
			连接断开之后，对服务器资源不会占用

		缺点：
			创建连接需要消耗时间

		95%以上都是 http 或是 https

	WebSocket
		长连接
		优点：
			发送消息快

		缺点：
			可能会对服务器资源造成占用
			心跳包

		应用场景：即时通讯 直播 游戏
